# 植物病害识别后的处置建议

## 重要说明

本平台负责图像识别和结果整理，不替代农技人员、实验室检测或当地植保部门的诊断。模型输出只能作为排查线索，不能仅凭模型类别直接决定用药。

## 建议的判断流程

1. 检查原始图片是否清晰，确认叶片正反面、病斑边缘和整体植株状态是否可见。
2. 记录作物种类、品种、生育期、种植地点、发病时间和近期天气。
3. 结合检测类别、置信度和标注图进行人工复核。
4. 对疑似病株和健康植株分别取样，避免采样过程造成交叉传播。
5. 结合当地农业技术部门或植保专家意见确认病害。
6. 只有在确认病害和作物适用范围后，才依据当地登记农药标签选择防治措施。

## 综合防治原则

- 优先使用清洁田园、合理密植、通风降湿、轮作、清除病残体和健康种苗等农业措施。
- 发现疑似病株后及时隔离和观察，避免人员、工具和灌溉水造成传播。
- 结合天气和田间调查进行预防，不应只根据单张图片盲目施药。
- 使用农药时必须遵守当地登记标签、适用作物、稀释倍数、安全间隔期和个人防护要求。
- 不建议把多个药剂随意混配，也不建议使用未登记或来源不明的产品。

## 如何理解模型结果

- 高置信度结果仍需要结合症状和环境复核。
- 低置信度结果可以作为进一步观察线索，但不应直接当作确诊结论。
- 健康叶片类别表示图片外观更接近训练集中的健康样本，不代表没有其他问题。
- 如果图片中的病斑不在训练类别范围内，模型可能返回相近类别或没有有效结果。

## 资料边界

本文只提供通用的风险控制和信息收集建议，不提供具体农药名称、剂量或针对单一病害的处方。具体防治方案应以当地农业主管部门、植保站、农技专家和农药登记标签为准。
