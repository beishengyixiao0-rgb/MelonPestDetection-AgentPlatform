#### API 记录规范

**后端成员新增或修改 API 时，必须记录以下内容**：

- 接口名称
- 请求方法和路径
- 是否需要登录鉴权
- 请求参数
- 响应数据
- 错误状态码
- 前端调用建议

示例：

- API：方法名称（如获取当前用户信息）
- 方法：GET
- 路径：如 /api/auth/me
- 鉴权：需要 `Authorization: Bearer <token>`/不需要
- 请求参数：无
- 后端文件：`backend/app/api/xxx.py`
- 前端封装：`frontend/src/api/xxx.js` 的 `xxxApi()`
- 调用页面/组件：`frontend/src/views/XxxPage.vue`
- 成功响应：

```json
{
  "id": 1,
  "username": "testuser",
  "email": "test@example.com",
  "roles": ["user"]
}
```

- 错误响应

```json
{
  "code": 400,
  "message": "错误信息",
  "data": null
}
```

- 前端调用示例

```js
import { xxxApi } from "@/api/xxx";
const res = await xxxApi(params);
```

#### API 记录

**1. 忘记密码**

- 方法：POST
- 路径：`/api/auth/forgot-password`
- 鉴权：不需要
- 请求参数：`{"email": "string"}`
- 后端文件：`backend/app/api/auth.py`
- 成功响应：`{"message": "重置令牌已生成", "token": "xxx", "expires_in": "1小时"}`
- 错误响应：`{"code": 404, "message": "该邮箱未注册", "data": null}`

**2. 重置密码**

- 方法：POST
- 路径：`/api/auth/reset-password`
- 鉴权：不需要
- 请求参数：`{"token": "string", "new_password": "string"}`
- 后端文件：`backend/app/api/auth.py`
- 成功响应：`{"message": "密码重置成功"}`
- 错误响应：`{"code": 400, "message": "无效的重置令牌/重置令牌已过期", "data": null}`

**3. 获取个人信息**

- 方法：GET
- 路径：`/api/auth/profile`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：无
- 后端文件：`backend/app/api/auth.py`
- 成功响应：

```json
{
  "id": 1,
  "username": "testuser",
  "email": "test@example.com",
  "phone": "13800138000",
  "avatar": null,
  "is_active": true,
  "roles": ["user"],
  "last_login_at": "2026-07-09T15:00:00",
  "created_at": "2026-07-09T14:00:00",
  "detection_stats": {
    "total_tasks": 0,
    "total_images": 0,
    "total_objects": 0,
    "avg_inference_time": 0.0,
    "class_distribution": {},
    "daily_trend": [],
    "scene_distribution": {}
  }
}
```

**4. 修改个人信息**

- 方法：PUT
- 路径：`/api/auth/profile`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：`{"phone": "string", "avatar": "string", "email": "string"}`（只传需修改的字段）
- 后端文件：`backend/app/api/auth.py`
- 成功响应：同获取个人信息
- 错误响应：`{"code": 400, "message": "该邮箱已被其他用户使用", "data": null}`

**5. 修改密码**

- 方法：PUT
- 路径：`/api/auth/password`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：`{"old_password": "string", "new_password": "string"}`
- 后端文件：`backend/app/api/auth.py`
- 成功响应：`{"message": "密码修改成功"}`
- 错误响应：`{"code": 400, "message": "旧密码错误", "data": null}`
