**4. 上传数据集压缩包**

- 方法：POST
- 路径：`/api/dataset/{dataset_id}/upload/zip`
- 鉴权：不需要
- 请求参数：`zip_file: UploadFile (.zip文件)`
- 后端文件：`backend/app/api/dataset.py`
- 支持压缩包结构：
  - 标准结构：`train/images/*.jpg`, `train/labels/*.txt`, `val/images/*.jpg`, `val/labels/*.txt`, `test/images/*.jpg`, `test/labels/*.txt`
  - 简单结构：`images/*.jpg`, `labels/*.txt`（默认归入train集）
- 成功响应：

```json
{

  "message": "success",
  "data": {
    "dataset_id": 1,
    "uploaded_images": 100,
    "parsed_labels": 95,
```
