#### API 记录

**1. 快捷单图检测**

- 方法：POST
- 路径：`/api/detection/single`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：`multipart/form-data`，`file: UploadFile`；可选 `conf: float`、`iou: float`、`scene_id: int`
- 后端文件：`backend/app/api/detection.py`
- 前端封装：`frontend/src/api/detection.js` 的 `detectSingle()`
- 调用页面/组件：`frontend/src/views/ChatPage.vue`
- 成功响应：`task_id`、`filename`、`total_objects`、`detections`、`class_counts`、`annotated_image_base64`、`inference_time`
- 错误响应：`400`（文件格式、阈值、模型或场景错误）、`401`（未登录）

**2. 快捷批量图片检测**

- 方法：POST
- 路径：`/api/detection/batch`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：`multipart/form-data`，多个 `files: UploadFile`；可选 `conf: float`、`iou: float`、`scene_id: int`
- 后端文件：`backend/app/api/detection.py`
- 前端封装：`frontend/src/api/detection.js` 的 `detectBatch()`
- 调用页面/组件：`frontend/src/views/ChatPage.vue`
- 成功响应：批量任务汇总及 `images` 图片检测结果列表
- 错误响应：`400`（未上传图片、文件格式、阈值、模型或场景错误）、`401`（未登录）

**3. 快捷 ZIP 检测**

- 方法：POST
- 路径：`/api/detection/zip`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：`multipart/form-data`，`file: UploadFile`（ZIP）；可选 `conf: float`、`iou: float`、`scene_id: int`
- 后端文件：`backend/app/api/detection.py`
- 前端封装：`frontend/src/api/detection.js` 的 `detectZip()`
- 调用页面/组件：`frontend/src/views/ChatPage.vue`
- 成功响应：ZIP 内图片的批量任务汇总及 `images` 图片检测结果列表
- 错误响应：`400`（非 ZIP、压缩包内无图片、阈值、模型或场景错误）、`401`（未登录）

**4. 查询检测任务状态**

- 方法：GET
- 路径：`/api/detection/status/{task_id}`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：路径参数 `task_id: int`
- 后端文件：`backend/app/api/detection.py`
- 前端封装：`frontend/src/api/detection.js` 的 `getDetectionStatus()`
- 调用页面/组件：`frontend/src/views/ChatPage.vue`
- 成功响应：`task_id`、`status`、`task_type`、`total_images`、`total_objects`、`completed_at`、`created_at`
- 错误响应：`404`（任务不存在或不属于当前用户）、`401`（未登录）

**5. 上传对话附件**

- 方法：POST
- 路径：`/api/chat/upload`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：`multipart/form-data`，`file: UploadFile`（图片或 ZIP）。Agent 多图场景由前端对每张图片分别调用本接口
- 后端文件：`backend/app/api/chat.py`
- 前端封装：`frontend/src/views/ChatPage.vue` 的附件上传逻辑
- 调用页面/组件：`frontend/src/views/ChatPage.vue`
- 成功响应：`{"image_path": "系统临时文件路径"}`；多图场景汇总多个 `image_path` 为 `image_paths`
- 错误响应：`400`（不支持图片或 ZIP 以外的文件）、`401`（未登录）

**6. 智能对话流**

- 方法：POST
- 路径：`/api/chat/stream`
- 鉴权：需要 `Authorization: Bearer <token>`
- 请求参数：JSON：`message: string`；可选单附件 `image_path: string`，或多图附件 `image_paths: list[string]`，以及 `session_id: int`、`scene_id: int`。`image_path` 与 `image_paths` 不能同时传入，且所有临时路径必须位于上传目录
- 后端文件：`backend/app/api/chat.py`
- 前端封装：`frontend/src/utils/stream.js` 的 SSE 解析逻辑
- 调用页面/组件：`frontend/src/views/ChatPage.vue`
- 成功响应：`text/event-stream`，按顺序返回 `text_chunk`、`tool_call`、`tool_result`，结束标志为 `[DONE]`。多图附件会触发 `detect_batch_images(image_paths=[...])`
- 错误响应：`400`（消息为空或附件路径无效）、`401`（未登录）；Qwen API Key 权限不足时流中返回 `error` 事件
