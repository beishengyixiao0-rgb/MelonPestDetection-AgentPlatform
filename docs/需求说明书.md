## 1. 项目背景与目标

### 1.1 项目背景

甜瓜是重要经济作物，其种植是特色农业的核心板块。然而，田间蚜虫、潜叶蝇等虫害具有体型微小、密集分布、叶片遮挡、田间背景复杂等特征。

**传统人工巡查方式存在严重痛点：**

- **效率低、成本高**：大面积瓜田巡检耗时久，微小害虫肉眼难以分辨。
- **预警滞后**：无法实时统计虫口密度，虫害爆发后才能发现，错失最佳防治窗口期。
- **决策缺失**：仅识别虫害种类，无法自动匹配对应防治方案，易导致农药滥用。
- **工具碎片化**：数据、训练、推理、记录分散，无一体化平台。
- **技术门槛高**：农业从业者难以完成数据集处理和模型调参。

### 1.2 项目目标

| 层面 | 目标 |
| :--- | :--- |
| **数据层面** | 完成甜瓜虫害数据集的导入、清洗、可视化、划分与增广，适配YOLOv11训练格式。 |
| **模型层面** | 基于数据集完成YOLOv11模型训练、评估与部署，实现高精度检测。 |
| **平台层面** | 搭建前后端分离的智能平台，实现检测、训练、历史、看板、AI对话全流程可视化操作。 |
| **业务层面** | 为种植户、植保人员提供虫害识别、统计与防治建议，落地智慧农业场景。 |

### 1.3 适用用户群体

- 规模化甜瓜/哈密瓜种植基地管理员
- 农业植保技术人员与科研院校研究者
- 智慧农业设备运维与巡检无人机操作员

## 3. 数据集需求分析

### 3.1 数据集来源

- **数据集名称**：Melon Cantaloupe Pest Dataset
- **来源**：Roboflow Universe 开源数据集
- **下载地址**：https://universe.roboflow.com/trisha-mae/melon-cantaloupe-pest/dataset/12
- **任务类型**：Object Detection (目标检测)
- **标注方式**：Bounding Box (边界框)


### 4.2 数据集基础参数

- **总图像数量**：6500 张甜瓜田间实拍图
- **数据集划分**：
  - 训练集：87% 
  - 验证集：8% 
  - 测试集：4% (待下载后重新确认)
- **预处理规则**：自动校正图像方向、统一缩放至 640×640、自适应对比度均衡。
- **数据增强策略**：90°旋转、亮度 ±15% 扰动，每样本生成3张增强图像。

### 4.3 数据类别

系统需支持识别以下 **5 类** 甜瓜核心害虫：

| 编号 | 类别 (英文) | 中文名称 |
| :--- | :--- | :--- |
| 0 | Aphids | 蚜虫 |
| 1 | Cutworms | 切根虫 |
| 2 | Leafminers | 潜叶蝇 |
| 3 | Red-Melon-Beetle | 红瓜甲虫 |
| 4 | Whiteflies | 白粉虱 |

### 4.4 数据处理需求

- 数据清洗与标签检查
- 统一图片尺寸 (640×640)
- 数据增强配置 (旋转、亮度等)
- 格式转换为 YOLO 标准格式

## 5. 系统总体需求

### 5.1 系统架构

系统采用前后端分离架构，前端负责用户交互与结果展示，后端提供API服务与模型推理。

用户层：
- 种植基地管理员
- 植保技术人员
- 农业科研人员
- 设备运维人员

前端展示层（Vue3 + Vite）：
- 检测工作台
- 训练工作台
- 数据看板
- AI对话
- 历史记录
- 数据集管理
- 用户管理
- 系统布局

API网关层（FastAPI）：
- 认证接口
- 检测接口
- 训练接口
- 对话接口

业务服务层：
- Detection Service（检测服务）
- Training Service（训练服务）
- Agent Service（智能体服务）
- User Service（用户服务）
- History Service（历史记录服务）
- MinIO Client（存储客户端）

数据存储层：
- PostgreSQL（业务数据）
- Redis（缓存与会话）
- MinIO（图片与模型文件）
- 向量数据库（RAG知识库）

AI模型层：
- YOLOv11 目标检测模型（训练、推理、权重管理、版本迭代）
- LLM 大语言模型（智能对话、防治建议生成、RAG检索）

### 5.2 数据流转流程

系统数据流转分为训练阶段和推理阶段两条主线。

训练阶段数据流：
1. Roboflow 甜瓜虫害数据集下载
2. 数据集上传至平台（图片 + YOLO标注）
3. 前端预处理配置（尺寸缩放、数据增强）
4. 后端启动 YOLOv11 训练任务
5. 训练过程实时监控（SSE 推送损失/精度）
6. 模型权重保存（best.pt 存入 MinIO）
7. 模型评估报告生成（混淆矩阵、各类AP）

推理阶段数据流：
1. 用户上传甜瓜图像/视频/摄像头流
2. 图像预处理（640×640 缩放、归一化）
3. YOLOv11 模型推理
4. 检测结果解析（类别、置信度、边界框）
5. 结果存储（PostgreSQL 记录 + MinIO 标注图）
6. 前端可视化展示（检测框叠加 + 虫口统计）
7. AI 对话生成防治建议（结合 RAG 知识库）

## 6. 系统功能需求

### 6.1 用户登录与权限管理模块

#### 6.1.1 注册功能

功能描述：用户通过用户名、邮箱、密码完成账号注册。
输入：用户名（3-50字符）、邮箱（有效格式）、密码（大于等于6位）。
处理逻辑：校验用户名/邮箱唯一性，密码加密存储，创建用户记录。
输出：注册成功返回用户信息（不含密码）。
异常处理：用户名已存在、邮箱已注册、参数格式错误返回400。

#### 6.1.2 登录功能

功能描述：用户通过账号密码登录系统。
输入：用户名、密码。
处理逻辑：验证用户名存在，校验密码，生成JWT Token。
输出：返回access_token、token_type、用户基本信息。
异常处理：用户名或密码错误返回401。

#### 6.1.3 路由权限守卫

功能描述：未登录用户无法访问核心业务页面。
处理逻辑：访问受保护路由时检查Token有效性，无效则跳转登录页。
输出：已登录正常访问，未登录跳转登录页并携带重定向地址。

#### 6.1.4 用户信息管理

功能描述：查看账号信息、退出登录、清空本地会话。
处理逻辑：退出时清除本地Token和用户信息。
输出：退出成功跳转登录页。

#### 6.1.5 角色权限预留

功能描述：区分普通农户与科研管理员。
权限差异：管理员开放模型批量管理、数据集批量导入等权限。

### 6.2 数据集管理模块

#### 6.2.1 数据集导入

功能描述：支持导入Roboflow下载的YOLO格式甜瓜虫害数据集。
输入：图片文件 + txt标注文件，或单文件压缩包。
处理逻辑：自动解析数据集划分（train/val/test），自动识别5类虫害标注标签。
输出：导入成功返回数据集概要信息（总图片数、各类别数量）。

#### 6.2.2 数据集可视化预览

功能描述：在线预览甜瓜田间样本图，叠加虫害标注框。
展示内容：类别名称、标注数量、总图片数、各类害虫样本分布柱状图。
用途：识别样本不均衡问题。

#### 6.2.3 数据预处理配置

功能描述：复用数据集原生预处理规则。
配置项：
- 640×640尺寸缩放
- 自适应对比度均衡
- 旋转角度配置
- 亮度扰动配置
- 增强倍数配置
- 脏数据清洗（过滤模糊、无标注、遮挡严重的图像）

#### 6.2.4 数据集导出

功能描述：支持导出标准YOLOv11训练格式。
输出：打包下载至本地。
版本管理：支持数据集版本保存，记录不同增广、清洗后的数据集快照。

### 6.3 YOLOv11 模型训练工作台

#### 6.3.1 训练参数可视化配置

基础参数：
- 批次大小（batch size）
- 迭代轮次（epochs）
- 学习率（learning rate）
- 置信度阈值（conf threshold）
- 数据集绑定（选择已导入的甜瓜虫害数据集）

高级参数：
- 注意力机制开关
- WIoU损失函数
- 多尺度训练（适配田间小害虫）

#### 6.3.2 训练实时监控

功能描述：SSE流式实时推送训练状态。
监控内容：
- 训练日志
- 损失曲线（box损失、分类损失、DFL损失）
- mAP50
- 精确率（Precision）
- 召回率（Recall）
- mAP50-95

训练控制：一键终止训练，保存当前断点权重。

#### 6.3.3 模型权重管理

功能描述：训练完成自动保存权重文件。
保存文件：best.pt（最优权重）、last.pt（最后一轮权重）。
列表展示：绑定对应甜瓜虫害数据集、训练时间、mAP精度。
操作功能：权重上传、删除、切换推理，支持多版本模型对比测试。

#### 6.3.4 训练结果评估

功能描述：基于数据集测试集自动生成评估报告。
评估内容：
- 混淆矩阵
- 各类虫害精确率柱状图
- 训练报告（含数据集信息、参数、精度指标、损失曲线）

### 6.4 虫害检测工作台

#### 6.4.1 多源图像输入

支持的输入方式：
- 单图上传：拖拽甜瓜叶片实拍图，实时检测
- 批量检测：一次性上传多张瓜田图片，批量生成检测结果
- 视频流检测：上传视频文件，逐帧检测
- 摄像头实时检测：模拟田间监控、无人机航拍画面实时推理

#### 6.4.2 可视化检测结果

功能描述：在原图上叠加检测信息。
显示内容：
- 虫害检测框
- 害虫类别标签
- 置信度数值
- 单张图片各类害虫数量统计
- 虫口密度统计

参数调节：支持调整置信度阈值，过滤低置信误检。

#### 6.4.3 结果导出与存储

功能描述：保存检测结果。
存储方式：
- 带标注检测图保存至本地
- 带标注检测图上传至MinIO
- 检测报表导出（CSV表格，记录每张图片虫害种类、数量、置信度）
- 检测记录自动存入历史数据库

### 6.5 历史记录管理模块
